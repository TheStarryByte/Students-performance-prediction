# Student Performance Prediction using Machine Learning

This project predicts students' exam scores based on their daily habits such as
study hours, sleep time, screen time, attendance, and lifestyle factors.

The goal is to explore how habits affect academic performance using data analysis
and machine learning techniques.

---

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---


## Results

The trained model predicts exam scores based on student habits and highlights
key factors that influence academic success.
